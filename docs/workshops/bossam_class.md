# Bossam (Boiled Pork) — Standalone Reference

A classic Korean boiled pork dish, often served with fresh kimchi and ssamjang for wrapping.

## Per 1 Couple

| Ingredient | Amount |
|------------|--------|
| Pork (shoulder/front leg, whole pork belly, or pork neck) | 200–300 g |
| Onion | 1/4 pc |
| Garlic cloves | 2–3 |
| Ginger | a little |
| Green onion | a little |
| Soybean paste (Doenjang) | 2 tbsp |
| Chili paste (Gochujang) | 1 tbsp |
| Sesame oil | 1 tbsp |
| Sugar | to taste |
| Salt | to taste |

## Per Session (7 couples / 14 people)

| Ingredient | Amount |
|------------|--------|
| Pork (belly, shoulder, or neck) | 1.4–2.1 kg |
| Onion | 2 pcs |
| Garlic | 30–40 cloves |
| Ginger | 2–3 bulbs |
| Green onion | 2–3 bunches |
| Soybean paste (Doenjang) | 1 cup (14 tbsp) |
| Chili paste (Gochujang) | 0.5 cup (7 tbsp) |
| Sesame oil | 0.5 cup (7 tbsp) |
| Sugar | 1 pack |
| Salt | 1 pack |

## Tools & Equipment (Per Couple)

- Large pot for boiling
- Cutting board
- Knife
- Tongs
- Serving platter
