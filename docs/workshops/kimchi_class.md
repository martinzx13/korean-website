# Kimchi Class — Fresh Kimchi (Geotjeori) + Bossam (Boiled Pork)

A hands-on workshop where couples learn to make two iconic Korean dishes from scratch.

---

## Fresh Kimchi (Geotjeori)

### Per 1 Couple

| Ingredient | Amount |
|------------|--------|
| Napa cabbage (Chinese/Asian Cabbage) | 1 pc |
| Garlic cloves | 2–3 |
| Fish sauce | 2 tbsp |
| Chili powder (Gochugaru) | 2–3 tbsp |
| Sugar | 1–2 tbsp |
| Sesame seeds | to taste |
| Green onion | a little |
| Ginger | a little |
| Salt | 2 tbsp |

### Per Session (7 couples / 14 people)

| Ingredient | Amount |
|------------|--------|
| Napa cabbage | 10 pcs |
| Garlic | 30–40 cloves |
| Fish sauce | 1 cup (14 tbsp) |
| Chili powder (Gochugaru) | 1.5–2 cups (14–21 tbsp) |
| Sugar | ~1 cup |
| Toasted sesame seeds | 1 pack |
| Green onion | 2–3 bunches |
| Ginger | 2–3 bulbs |
| Salt | ~1 cup |

---

## Bossam (Boiled Pork)

### Per 1 Couple

| Ingredient | Amount |
|------------|--------|
| Pork (shoulder/front leg, whole pork belly, or pork neck) | 200–300 g |
| Onion | 1/4 pc |
| Garlic cloves | 2–3 |
| Ginger | a little |
| Green onion | a little |
| Soybean paste (Doenjang) | 2 tbsp |
| Chili paste (Gochujang) | 1 tbsp |
| Sesame oil | 1 tbsp |
| Sugar | to taste |
| Salt | to taste |

### Per Session (7 couples / 14 people)

| Ingredient | Amount |
|------------|--------|
| Pork (belly, shoulder, or neck) | 1.4–2.1 kg |
| Onion | 2 pcs |
| Garlic | 30–40 cloves |
| Ginger | 2–3 bulbs |
| Green onion | 2–3 bunches |
| Soybean paste (Doenjang) | 1 cup (14 tbsp) |
| Chili paste (Gochujang) | 0.5 cup (7 tbsp) |
| Sesame oil | 0.5 cup (7 tbsp) |
| Sugar | 1 pack |
| Salt | 1 pack |

---

## Tools & Equipment (Per Couple)

- Cutting mat / Cutting board
- Knife
- Cooking gloves
- Big bowl and tray
- 2–3 spoons
